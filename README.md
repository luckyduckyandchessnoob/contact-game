# Contact! Digital version
Contact! is a folk game for three or more players. One player is the wordmaster, and the other players try to guess the word by giving each other possible clues. This is the digital version of the game for two or more players. Come back daily for a new word!
## How to play
1. One player starts a new game and shares the room code with the other players. Once everyone has entered the code, the host can start the game.
2. The first letter of the word will be revealed at the top of the screen. Players think of what the word could be.
